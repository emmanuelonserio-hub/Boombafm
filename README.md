#  Boomba FM Music Player

Boomba FM is a small console music player written in Python.  
It lets you organize songs in folders, list them, play, pause, resume, stop, and even search by keyword.

##  Project Setup

Folder structure should look like this:
Boombafm/
│
|--- main.py           
|--- music/ 
|--- README        

##  Requirements

- Python 3 installed
- Pygame library

Install pygame with:

```terminal
pip install pygame

##  Features

- List all songs → Shows every song saved in the music folder.  
- Play → Pick a song and start listening.  
- Pause / Resume / Stop → Control the music easily.  
- Search songs → Type a word like "Nairobi" or "Mazishi" to quickly find a track.  
- Exit → Close the player when you’re done.
